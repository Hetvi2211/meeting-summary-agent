# Code Review Checklist

## Security

- Hardcoded API keys
- SQL injection risks
- Unsafe eval()
- Sensitive data exposed

## Performance

- Unnecessary loops
- Duplicate API calls
- Large file reads
- Memory leaks

## Maintainability

- Meaningful variable names
- Modular functions
- Error handling
- Code duplication

## Readability

- Clear comments
- Consistent formatting
- Small functions
- Easy-to-follow logic
